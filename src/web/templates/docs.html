Azure RAG Financial System Documentation

Overview

This application provides a web interface for querying financial documents using Azure Cognitive Search and Azure OpenAI. It supports document ingestion, semantic search, and LLM-powered Q&A.

Deployment Steps

  1. Provision Azure resources: Web App, Cognitive Search, OpenAI, Storage Account, Key Vault.
  2. Configure environment variables for service credentials.
  3. Deploy the application code to Azure Web App.
  4. Run the scrapper and document ingestion scripts.

Running the Scrapper

python main.py rag --process --input-dir demo_filings

This command processes documents and uploads embeddings to Azure Cognitive Search.

Viewing Data in Vector Store

  • Go to Azure Portal → Cognitive Search → Search Explorer to view indexed documents.
  • Use the web app's query interface for semantic search and Q&A.
